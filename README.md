# Vanilla JS Lightbox

## How to use

Link `lightbox.js` and `lightbox.css` in your files.

The basic markup is:

```html
<div class="lightbox">
  <figure>
    <img src="/img/source" alt="image alt text" />
    <!-- figcaption is not required -->
    <figcaption>Image Description</figcaption>
  </figure>
</div>
```
